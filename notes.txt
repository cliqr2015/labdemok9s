==> Next steps:

    echo >> /home/k9s/.bashrc
    echo 'eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv bash)"' >> /home/k9s/.bashrc
    eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv bash)"
- Install Homebrew's dependencies if you have sudo access:
    sudo apt-get install build-essential

Quick Checks After Reboot

In terminal (Ctrl+Alt+F3 if needed):Bash
echo $XDG_SESSION_TYPE   # should say "wayland" or "x11" after login

To switch back to console mode temporarily:Bash
sudo systemctl isolate multi-user.target

To return to GUI:Bash
sudo systemctl isolate graphical.target
